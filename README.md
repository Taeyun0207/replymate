# ReplyMate Extension (Starter)

Minimal project scaffold for a Chrome (Manifest V3) extension that will assist with drafting replies in Gmail.

## Structure

- `manifest.json` — extension manifest (MV3)
- `content/gmail.js` — content script for `mail.google.com`
- `background/service-worker.js` — background service worker
- `popup/popup.html` — action popup UI
- `options/options.html` — options page
